# AI Text Summarizer

AI-powered text summarizer built with React.js using OpenRouter API.

## Features
- Short, Medium & Long summaries
- Copy to clipboard
- Download summary as PDF
- Word & character counter
- Responsive UI
- 5000 character limit validation

## Tech Stack
- React.js
- Tailwind CSS
- Lucide React
- jsPDF
- OpenRouter API
